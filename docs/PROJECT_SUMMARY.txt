QUICK START
================================================================================

1. CREATE .env FILE
   ─────────────────
   Create a file named ".env" in the project root with:

   OPENAI_API_KEY=sk-your-key-from-https://platform.openai.com/account/api-keys

2. START THE APPLICATION
   ──────────────────────
   Windows PowerShell:
   $ .\run.ps1

   $ ./run.sh

   ──────────────────
   Visit: http://localhost:5173
